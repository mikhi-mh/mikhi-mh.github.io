  • Task Management REST API

    • (Domain: Backend Development | Java , Spring Boot) | Source-Code : GitHub-Link | Hosted App

    Details:

    A simple but powerful RESTful TODO-app / Task Management API built with Spring Boot, Spring Data JPA, and
    H2 in-memory database. It supports full CRUD functionality, filtering by task status and due date, pagination, and sorting. It includes global exception handling, validation, and is documented using Swagger/OpenAPI.

    Key Features:

    • CRUD operations for tasks ( Create, Read, Update, Delete )
    • Pagination, Sorting, and filtering (status, due date),
    • Global exception handling & validation for Fields/Entites
    • H2 database with sample data, Flyway Migration and accessible console
    • Swagger UI & OpenAPI docs
    • Dockerized for deployment ease
    • Unit tests & integrations tests for core functionality
    • Hosted using Render (free tier).
